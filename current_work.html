Black carbon source apportionment project

This is currently my main project. Due to difficulties in measuring black carbon (BC) concentartions, we simulate emissions using the Lagrangian Particle Dispersion model FLEXPART (version 10.4). This helps us better understand sources in order to mitigate them.

Controlling FLEXPART with Python

A task that can be quite tedious is made easy with this small project. Sometimes we wish to run multiple simulations (not to be confused with multiple releases), and we don't want to manually change input settings and what not. In this repository, you can find a script that will prepare simulation input, and run the model for you! All you have to provide is a list of information (see README)! It was made to be flexible so that the user can add as many changes as they please. All is written in Python with OOP-style.
